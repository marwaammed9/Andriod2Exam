## Description
Android application with secure authentication and swipeable tabbed UI.

## Features
- Email and password validation with TextWatchers
- Double back press to exit with Snackbar
- TabLayout with ViewPager2 (3 tabs)
- Custom back stack navigation
- PopupMenu with Light/Night/System modes
- Ancestral Navigation (Up Button)

## Tech Stack
- Kotlin
- ViewBinding
- ViewPager2
- TabLayout
- Material Design
